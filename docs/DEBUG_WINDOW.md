# Debug窗口使用说明

## 功能概述

Debug窗口是一个独立的调试控制台，用于实时查看程序的所有调试信息。

## 如何打开

在主窗口的工具栏中，点击"调试窗口"按钮即可打开debug控制台。

## 功能特性

- 实时显示程序运行日志
- 支持多种日志级别：INFO、WARN、ERROR、DEBUG
- 自动滚动到最新日志（当滚动到底部时）
- 可以手动清空日志记录
- 每条日志包含时间戳、级别和消息内容

## 开发者使用

在Rust代码中，可以使用以下宏来记录日志：

```rust
// 信息日志
debug_info!("这是一条信息日志");

// 警告日志
debug_warn!("这是一条警告日志");

// 错误日志
debug_error!("这是一条错误日志");

// 调试日志
debug_debug!("这是一条调试日志");
```

这些宏会同时：
1. 输出到tracing日志系统（写入日志文件）
2. 发送到debug窗口实时显示

## 界面说明

- 顶部显示"Debug Console"标题
- 右上角有"清空"按钮，可以清除所有日志
- 主体区域显示日志列表，每条日志包含：
  - 时间戳（精确到毫秒）
  - 日志级别（不同颜色区分）
  - 日志消息内容
