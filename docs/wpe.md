# 说明游戏中的操作与封包的对应关系

## 本文档内的变量说明

**Note: 该部分应用的格式为{var}**

|**变量名**|**意义**|**注释**|
|:---|:---:|:---|
|`hex_qq_num`|玩家QQ号的16进制表示(共$8$位)|不需要带`0x`|
|`spirit_pos`|宠物在背包内的次序|$spirit\_pos\in [1, 6]$|
|`map_no`|地图编号|-|

|封包格式|对应操作|
|:---|:---|
|95270000000B0014`{hex_qq_num}`00000000000000010`{spirit_pos}`|把宠物背包的第`{spirit_pos}`位放入仓库|
|9527000000030003`{hex_qq_num}`000000000000000200`{map_no}`|跳转到编号为`map_no`的场景|
|System_宠物逃跑|从一场战斗中逃跑|
|95270000000B0052`{hex_qq_num}`00000000000000010`{spirit_pos}`|把宠物背包的第`{spirit_pos}`位放到家园锻炼|
